# Performance Enforcer - Example Next.js App

This is a sample Next.js application configured with Performance Enforcer.

## Setup

```bash
# Install dependencies
npm install

# Build the application
npm run build

# Run performance checks
npx performance-enforcer check

# Launch dashboard
npx performance-enforcer dashboard
```

## Performance Configuration

This app includes `.performance-enforcer.yml` with sensible defaults for a typical Next.js application.

## Features

- Home page with static content
- Dashboard route with charts
- About page
- Responsive design
- Server-side rendering

## Next.js Version

- Next.js 15
- App Router
- React 19
